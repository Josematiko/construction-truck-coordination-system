# 🚚 Construction Truck Coordination System

A web-based platform that connects customers with verified truck owners for construction material deliveries. The system simplifies the process of requesting transport services, assigning drivers, tracking deliveries, and managing construction logistics.

## ✨ Features

- Customer registration and login
- Truck owner registration and verification
- Driver management and assignment
- Multiple truck management
- Construction material delivery requests
- Order tracking
- Customer and truck owner messaging
- M-Pesa payment integration
- Review and rating system
- Administrative dashboard
- Reports and analytics

## 👥 System Users

- Customer
- Truck Owner
- Driver
- Administrator

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Josematiko/construction-truck-coordination-system.git
```

Navigate to the project:

```bash
cd construction-truck-coordination-system
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python run.py
```
